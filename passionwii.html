Plugin Passion Wii Streaming para Movian

Passion Wii Streaming es una plataforma multipantalla para Wii, Wii U, PS3 y DangoPlayer (Android, Android TV y Google TV), en la Cual Podras disfrutar De Peliculas, Series, Animes, Doramas, y mucho mas.

Este es un plugin para el reproductor multimedia Movian, no es una aplicacion independiente.
Necesitarás tener instalado Movian en tu PS3 para poder usar este plugin.
